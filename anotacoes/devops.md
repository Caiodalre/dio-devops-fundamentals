# DevOps

## O que é DevOps?

DevOps é uma cultura e um conjunto de práticas que promove a integração entre as equipes de desenvolvimento e operações.

O objetivo é melhorar a colaboração, automatizar processos e acelerar a entrega de software com qualidade e estabilidade.

## Objetivos do DevOps

- Acelerar o tempo de entrega do software
- Melhorar a comunicação entre equipes
- Reduzir falhas em produção
- Automatizar processos repetitivos
- Aumentar a confiabilidade dos sistemas
- Melhorar a experiência do usuário final

## Dev e Ops

### Dev

O profissional Dev é responsável por desenvolver e implementar funcionalidades nos sistemas.

### Ops

O profissional Ops é responsável por garantir estabilidade, segurança, escalabilidade e disponibilidade dos sistemas e da infraestrutura.

## CALMS

CALMS é um modelo utilizado para representar os pilares do DevOps:

- Culture: Cultura
- Automation: Automação
- Lean: Mentalidade enxuta
- Measurement: Medição
- Sharing: Compartilhamento

## As 3 Maneiras do DevOps

As 3 maneiras do DevOps são:

1. Fluxo
2. Feedback
3. Aprendizado contínuo

Esses princípios ajudam a melhorar a entrega de software e promover evolução contínua nos processos.
